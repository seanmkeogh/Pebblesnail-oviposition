This page serves as a data repository for the molecular analyses performed in Keogh, Celli, and Ramos 202X: Ecological interactions between imperiled molluscs: Somatogyrus pebblesnail oviposition on freshwater mussels

1. The 'epibionts.phy' file is the primary alignment file used in IQ-TREE analyses
2. The 'modelfinder' folder contains output for ModelFinder (finding the best-fit substitution model)
3. The 'p-distances' folder contains input .phy file and R script to calculate p-distances with .csv files of output
4. The 'tree' folder contains output for IQ-TREE phylogenetic analysis
